To download the Assignment:
git clone the repository under AdamOlivieri/Assignment2 to your local machine

To compile: Within Intellij and opening the project Assignment_1, start by
running Server to check compiling.

To run: After clicking the run button on Server, run the Client. This will 
start the ui. To upload, select a file under Client and press the upload 
button. To download, select a file under Server and press the download
button.
